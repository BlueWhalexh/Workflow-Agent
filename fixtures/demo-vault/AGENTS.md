# Demo Vault Agent Instructions

This vault follows the LLM Wiki three-layer methodology.

- Treat `raw/` as read-only source material.
- Write structured knowledge only under `knowledge-base/`, `daily/`, `projects/`, or `resources/`.
- Do not edit `schema/` without explicit `schema-edit` approval.
- Every `knowledge-base/` change must update `knowledge-base/index.md` when links change and append a note to `log.md`.
- Follow `schema/CLAUDE.md`, `schema/methodology.yaml`, and `schema/ruleset.yaml`.
